Radiofav

Built using Laravel, HTML, CSS, JS, Bootstrap

I chose to build Radiofav using the Laravel framework, so I could understand how this framework works.

Radiofav enables you to bookmark a song you are currently listening to, if you're using one of the services it supports. Think of Radiofav like Pocket, but for songs. Thanks to this project I've become quite familiar with Laravel and made me understand how important it is to use a framework.
